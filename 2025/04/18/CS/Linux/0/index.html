з¬ҰеҸ·жЁЎејҸ(Symbolic Mode)

дҪҝз”Ёеӯ—жҜҚе’Ңз¬ҰеҸ·жқҘиЎЁзӨәжқғйҷҗпјҡ

  • uпјҡжүҖжңүиҖ…
  • gпјҡз»„
  • oпјҡе…¶д»–дәә
  • aпјҡжүҖжңүз”ЁжҲ·(u,g,o)
  • +пјҡж·»еҠ жқғйҷҗ
  • -пјҡз§»йҷӨжқғйҷҗ
  • =пјҡи®ҫзҪ®жқғйҷҗ

дҫӢеҰӮпјҡ

  • u+rпјҡз»ҷжүҖжңүиҖ…ж·»еҠ иҜ»еҸ–жқғйҷҗ
  • g-wпјҡз§»йҷӨз»„зҡ„еҶҷе…Ҙжқғйҷҗ
  • o=rxпјҡе°Ҷе…¶д»–дәәзҡ„жқғйҷҗи®ҫзҪ®дёәиҜ»еҸ–е’Ңжү§иЎҢ
  • a+wпјҡз»ҷжүҖжңүз”ЁжҲ·ж·»еҠ еҶҷе…Ҙжқғйҷҗ

ж•°еӯ—жЁЎејҸ(Numeric Mode)

дҪҝз”Ёж•°еӯ—жқҘиЎЁзӨәжқғйҷҗгҖӮжҜҸдёӘжқғйҷҗзұ»еһӢ(r,w,x)еҜ№еә”дёҖдёӘж•°еӯ—пјҡ

  • иҜ»еҸ–(r)пјҡ4
  • еҶҷе…Ҙ(w)пјҡ2
  • жү§иЎҢ(x)пјҡ1
  • жІЎжңүжқғйҷҗпјҡ0

е°ҶжүҖжңүиҖ…гҖҒз»„е’Ңе…¶д»–дәәзҡ„жқғйҷҗж•°еӯ—зӣёеҠ пјҢеҫ—еҲ°дёҖдёӘдёүдҪҚж•°зҡ„ж•°еӯ—

дҫӢеҰӮпјҡ

  • 7пјҡrwx (4 + 2 + 1)пјҡиҜ»еҸ–гҖҒеҶҷе…ҘгҖҒжү§иЎҢгҖҒ
  • 6пјҡrw- (4 + 2 + 0)пјҡиҜ»еҸ–гҖҒеҶҷе…Ҙ
  • 5пјҡr-x (4 + 0 + 1)пјҡиҜ»еҸ–гҖҒжү§иЎҢ
  • 4пјҡr-- (4 + 0 + 0)пјҡиҜ»еҸ–
  • 3пјҡ-wx (0 + 2 + 1)пјҡеҶҷе…ҘгҖҒжү§иЎҢ
  • 2пјҡ-w- (0 + 2 + 0)пјҡеҶҷе…Ҙ
  • 1пјҡ--x (0 + 0 + 1)пјҡжү§иЎҢ
  • 0пјҡ--- (0 + 0 + 0)пјҡжІЎжңүжқғйҷҗ

дҫӢеҰӮпјҢж•°еӯ—755иЎЁзӨәпјҡ

  • жүҖжңүиҖ…пјҡrwx (7)
  • з»„пјҡr-x (5)
  • е…¶д»–дәәпјҡr-x (5)

жҹҘзңӢжқғйҷҗ

дҪҝз”Ёls -l <ж–Үд»¶еҗҚ>жҹҘзңӢж–Үд»¶жқғйҷҗпјҡ

з¬¬дёҖдёӘеӯ—з¬Ұ-иЎЁзӨәиҜҘж–Үд»¶зұ»еһӢдёәжҷ®йҖҡж–Үд»¶пјҢеҗҺйқўзҡ„еӯ—з¬ҰжҳҜжқғйҷҗиҜҙжҳҺ

д»ҘдёӢжҳҜдёҖдәӣеёёи§Ғзҡ„ж–Үд»¶зұ»еһӢпјҡ

  • -пјҡжҷ®йҖҡж–Үд»¶ (Regular file)пјҡиҝҷжҳҜжңҖеёёи§Ғзҡ„ж–Үд»¶зұ»еһӢпјҢеҢ…еҗ«дәҶж•°жҚ®пјҲж–Үжң¬гҖҒдәҢиҝӣеҲ¶зӯүпјүгҖӮ
  • dпјҡзӣ®еҪ• (Directory)пјҡзӣ®еҪ•жҳҜз”ЁжқҘз»„з»Үе’ҢеӯҳеӮЁе…¶д»–ж–Үд»¶е’Ңзӣ®еҪ•зҡ„зү№ж®Ҡж–Үд»¶гҖӮ
  • lпјҡз¬ҰеҸ·й“ҫжҺҘ (Symbolic link)пјҡз¬ҰеҸ·й“ҫжҺҘпјҲд№ҹз§°дёәиҪҜй“ҫжҺҘпјүжҳҜжҢҮеҗ‘еҸҰдёҖдёӘж–Үд»¶жҲ–зӣ®еҪ•зҡ„зү№ж®Ҡж–Үд»¶гҖӮе®ғзұ»дјјдәҺ Windows дёӯзҡ„еҝ«жҚ·ж–№ејҸгҖӮ
  • bпјҡеқ—и®ҫеӨҮж–Үд»¶ (Block device file)пјҡеқ—и®ҫеӨҮж–Үд»¶з”ЁдәҺи®ҝй—®еқ—и®ҫеӨҮпјҢеҰӮзЎ¬зӣҳй©ұеҠЁеҷЁгҖҒCD-ROM й©ұеҠЁеҷЁзӯүгҖӮе®ғд»¬д»Ҙеӣәе®ҡеӨ§е°Ҹзҡ„еқ—иҝӣиЎҢж•°жҚ®дј иҫ“гҖӮ
  • cпјҡеӯ—з¬Ұи®ҫеӨҮж–Үд»¶ (Character device file)пјҡеӯ—з¬Ұи®ҫеӨҮж–Үд»¶з”ЁдәҺи®ҝй—®еӯ—з¬Ұи®ҫеӨҮпјҢеҰӮз»Ҳз«ҜгҖҒжү“еҚ°жңәзӯүгҖӮе®ғд»¬д»Ҙеӯ—иҠӮжөҒзҡ„ж–№ејҸиҝӣиЎҢж•°жҚ®дј иҫ“гҖӮ
  • pпјҡе‘ҪеҗҚз®ЎйҒ“ (Named pipe)пјҡе‘ҪеҗҚз®ЎйҒ“пјҲд№ҹз§°дёә FIFOпјүжҳҜдёҖз§Қзү№ж®Ҡзҡ„ж–Үд»¶пјҢз”ЁдәҺиҝӣзЁӢй—ҙйҖҡдҝЎгҖӮж•°жҚ®еҶҷе…Ҙз®ЎйҒ“зҡ„дёҖз«ҜпјҢд»ҺеҸҰдёҖз«ҜиҜ»еҸ–гҖӮ
  • sпјҡеҘ—жҺҘеӯ—ж–Үд»¶ (Socket file)пјҡеҘ—жҺҘеӯ—ж–Үд»¶з”ЁдәҺиҝӣзЁӢй—ҙйҖҡдҝЎпјҢйҖҡеёёз”ЁдәҺзҪ‘з»ңйҖҡдҝЎгҖӮ
